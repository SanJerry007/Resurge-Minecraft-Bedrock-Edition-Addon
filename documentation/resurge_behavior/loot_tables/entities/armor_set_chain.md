# armor_set_chain

每个部件的装备分别进行抽取，共抽取4次。



## 每次抽取条件

与难度相关的随机概率。

| 和平 | 简单 | 普通 | 困难 |
| ---- | ---- | ---- | ---- |
| 0%   | 50%  | 50%  | 75%  |
